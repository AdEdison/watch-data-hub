# Watch Data Hub

Central CSV data hub for watch market data.

## Public CSV

- `data/watch_market_snapshot.csv`

## Columns

- `watch_id`
- `brand`
- `model`
- `reference_number`
- `display_name`
- `year`
- `production_years`
- `movement`
- `case_material`
- `case_size`
- `condition`
- `market_price_usd`
- `market_price_source`
- `market_price_updated_at`
- `auction_price_usd`
- `auction_house`
- `auction_date`
- `auction_lot_url`
- `shop_sources`
- `source_count`
- `source_urls`
- `data_quality`
- `notes`
- `updated_at`

## Data notes

Market prices are indicative and may be based on active listings, historical listings, auction results, or manual overrides.
